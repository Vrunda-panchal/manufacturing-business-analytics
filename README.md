# Manufacturing Business Analytics Dashboard

Business analytics project built using **Python, MySQL, and Power BI** to analyze customer sales, vendor performance, purchases, and product-level insights for a manufacturing company.

## Tools Used
- Python (Pandas)
- MySQL
- Power BI
- Jupyter Notebook
- Excel

## Key Features
- Customer Revenue Analysis
- Vendor Performance Analysis
- Top Selling Products
- Purchase vs Sales Comparison
- Average Customer Bill Value
- Interactive Power BI Dashboard

## Workflow
Excel Data → Python → MySQL → Power BI → Business Insights

## Dashboard Insights
- Top Customers by Revenue
- Vendor Spend Distribution
- Best Selling Products
- Business KPI Tracking
